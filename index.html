Tester Quick Guide

Use this checklist before sending feedback.

  1. Open the app and confirm all sections render without errors.
  2. Create one STP with at least two strata, then save the STP.
  3. Upload a map image and use View Map Full Screen to open a full reference view.
  4. Use Save Project and Load Project to confirm project persistence.
  5. Download XLSX and CSV exports and confirm both files are created.

Bug report format: browser and device, exact steps, expected result, actual result, and screenshot if available.
